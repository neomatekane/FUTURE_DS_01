The dataset that was used for this project is from Kaggle and consists of e-Commerce behavior data for November 2019. This data is extremely large and therefore special techniques were utilized. This project analyzes user behaviour to comprehend how users move through the market funnel from viewing products to then eventually buying them. 
The main objective of this project is to identify where users drop off in the funnel and look for stages that require attention and provide analysis to improve conversion rates.
Analysis was done using Python libraries such as pandas for data processing. seaborn and Matplotlib for advanced data visuals through Google Colab.
Since the dataset is extremely large , chunk processing was implemented to prevent system crashes.The marketing funnel was defined using three key stages based on user behavior: visitors were represented by “view” events, leads by “cart” events, and customers by “purchase” events.
This structure allowed for a clear analysis of user progression through the funnel. Key performance metrics calculated in this analysis include conversion rates between stages, specifically the transition from visitors to leads and from leads to customers, as well as drop-off rates indicating the proportion of users lost at each stage of the funnel.
A range of visualizations were created to support the analysis, including a funnel distribution chart showing the number of users at each stage, bar charts illustrating conversion and drop-off rates, category-level performance comparisons, event type distributions, price distributions, and time-based trends in user activity. These visualizations were generated using Seaborn to ensure clarity and professional presentation.
The analysis revealed a significant drop-off between the viewing and cart stages, indicating that while many users browse products, relatively few proceed to add items to their cart. 
In contrast, users who add items to their cart demonstrate a higher likelihood of completing a purchase. It was also observed that the majority of user interactions occur at the browsing stage, and that certain product categories exhibit stronger conversion performance, suggesting higher-quality leads within those segments.The primary area of concern within the funnel is the transition from visitors to leads, where the largest drop-off occurs. 
Product categories with higher lead-to-customer conversion rates were identified as sources of high-quality leads. Overall, the analysis indicates that the early stage of the funnel requires the most attention, although improvements in the later stages can further enhance overall performance.
To improve conversion rates, several strategies are recommended. Enhancing product pages with better images and descriptions can increase user engagement and encourage cart additions.
