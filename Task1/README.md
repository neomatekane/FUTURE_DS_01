
This dataset was taken from kaggle and it was cleaned and prepared for analysis using the following steps:
Removed duplicate records to ensure data accuracy
Handled missing values in the Postal Code column
Converted date columns into proper datetime format
Created new features such as Year, Month, and Shipping Days
Removed irrelevant columns (IDs) to improve analysis clarity
Ensured all sales values are valid (no zero or negative values)
In addition,From the analysing this retail sales data with this project I have managed to uncover business insights related to revenue trends, product performance, and regional sales.My source was Kaggle Superstore Dataset with 9800 rows and has features like Sales, Category, Region,Product and Dates. Insights from the data is the Sales show steady growth with seasonal peaks and a small number of products drives most revenue.Shipping effiecency varies across modes and regional differences highlight expnsion opportunities. Below are some of the recommendation a business can implement:
Focus marketing on top-performing products and improve underperforming areas by optimizing shipping processes,this will allow expansion in strong regions. An investment in high performing catergories could potentially improve sales.
